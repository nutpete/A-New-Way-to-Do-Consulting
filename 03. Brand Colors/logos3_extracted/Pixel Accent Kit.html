▶ PIXEL ACCENT KIT v1.0

A small system for pixel energy.

Eight elements that let the arcade aesthetic seep into the paper sections without taking over. Stepped shadows, square corners, Press Start 2P labels — applied to existing site components, not new ones.

▶ RULE 01 — SHADOWS
Replace box-shadow: 0 4px 12px with stacked Npx 0 0 + 0 Np 0 + Np Np 0 in the accent color.
▶ RULE 02 — SHAPES
No border-radius. Squares replace circles. Use crisp shape-rendering: crispEdges on SVG.
▶ RULE 03 — TYPE
Press Start 2P only on labels, badges, arrows. Never body. Never headlines.

▶ 02 / PIXEL BUTTON

Stepped shadow

Square corners, Press Start 2P label, fluency-3 stepped shadow. On hover, translates -2px and shadow grows.

box-shadow: 5px 5px 0 var(--fluency-3); :hover { transform: translate(-2px,-2px); box-shadow: 7px 7px 0 var(--fluency-3); }
// REPLACES hero "Book a time" + nav CTA
